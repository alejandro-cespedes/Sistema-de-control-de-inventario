Opciones

¿Que es el modulo de opciones?


El modulo de opciones permite cambiar la configuracion global del sistema, tales como el nombre del negocio, el logo, el mensaje personal y el iva a utilizar en las facturas y pedidos.


Solo el administrador puede acceder a este modulo, a su vez se puede descargar un respaldo de la base de datos e incluso recuperar la base de datos importando el archivo de respaldo descargado.

¿Cómo descargar la base de datos?


En la imagen se pueden ver 3 opciones, Importar base de datos,Descargar base de datos y Configuracion general.


Al escoger 'descargar base de datos' se descargara un archivo en formato XML con los datos actuales de la base de datos.



¿Cómo importar la base de datos?


En el siguiente formulario solo se tiene que escoger un archivo en formato XML que contenga la base de datos y luego, solo darle al boton de 'Ingresar'

Si se ha descargado antes un respaldo, este se podria usar para importar la base de datos que habia hasta ese punto, de usarse un archivo que este corrupto o sea invalido, se mostrara un mensaje de error.

Configuracion general



En la imagen se ven reflejado los campos del formulario, los campos son claros y autoexplicativos.


Una vez realizado los cambios deseados, solo se debe dar en 'Ingresar'.


Solo el administrador puede acceder a este modulo.
